# Линия за боядисване — подвески

Самостоятелен модул (прототип) за визуализиране на **конвейера за
боядисване** и **подвеските** върху него, с акцент върху **бързо отчитане**.
Замислен е после да се **слее** с основното приложение „Данко Системс“.

## Как се пуска

Отвори `boya/index.html` в браузъра. Не е нужна настройка — данните се
пазят локално в браузъра (localStorage).

## Какво прави

- **Конвейер с подвески** — подвеските висят на релса една след друга,
  както реално на линията.
- **На всяка подвеска:** цвят (RAL), вид детайли, брой детайли.
- **Бързо отчитане** — два статуса: *на линията* и *готово*. С едно
  докосване на бутона „✓ Готово“ подвеската се отчита.
- **Обобщение** отгоре: брой подвески и детайли на линията / готови.
- **Филтри:** всички · на линията · готови.
- **Редакция** — клик върху картата отваря прозорче за промяна/изтриване.
- **Премахни готовите** — изчиства приключилите подвески от линията.
- **Печат / справка** — за разпечатка на текущото състояние.
- Промените се отразяват **между отворените раздели** (наподобява realtime).

## Цвят по RAL

Често ползвани прахови RAL цветове имат вграден цвят за квадратчето.
Ако RAL-ът не е разпознат, избери цвят ръчно с пипетата до полето.

## Файлове

| Файл | Предназначение |
|------|----------------|
| `index.html` | Структура и интерфейс |
| `painting.css` | Стилове (същата палитра като основното приложение) |
| `painting.js` | Логика + слой за данните (`Store`) |
| `painting-setup.sql` | Готов SQL за Supabase (нужен само при сливане) |

## Сливане с основното приложение

Слоят за данните е изолиран в обекта `Store` в `painting.js`. За реално
споделяне на живо:

1. Пусни `painting-setup.sql` в Supabase (създава таблица `painting`).
2. Замени `Store` с обвивка около Supabase клиента (`list / upsert /
   remove / subscribe`). Форматът на записа
   (`{id, data, done, done_at, created_at, updated_at}`) вече съвпада с
   реда в таблицата, така че останалата логика остава непроменена.
3. По избор: добави бутон „🎨 Боядисване“ в хедъра на основното
   приложение, който отваря този изглед.
