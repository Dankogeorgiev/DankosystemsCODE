# Данко Системс — Следене на мостри

Малко уеб-приложение за следене на изпълнението на мострите в Данко Системс.
Работи изцяло в браузъра — **без сървър и без инсталация**. Данните се пазят
локално в браузъра (localStorage).

## Как се пуска

Отвори файла `index.html` с двойно кликане (или с десен бутон → „Отвори с“ →
браузър). Готово.

> Съвет: за да работи на повече от един компютър, използвай бутоните
> **Експорт** / **Импорт** за прехвърляне на данните чрез JSON файл.

## Какво съдържа всяка мостра

1. **Име на клиента**
2. **Информация за клиента** — свободно поле
3. **Информация за мострата** — свободно поле
4. **Чертежи за производство** — прикачване на изображения и PDF
5. **Необходими материали** — таблица (материал, количество, забележка)
6. **Поръчка на материали** — статус, доставчик, забележки
7. **Производствен процес** — тикче + отговорник за всяка операция:
   - Лазерно рязане
   - Огъване
   - Заварване
   - Допълнителни операции
   - Боядисване
   - Сглобяване
8. **Анализ на производството** — какво е направило впечатление

## Функции

- Списък с всички мостри + търсене
- **Краен срок** на мострата и **дата на поръчката** / очаквана доставка
- **Статус „Завършена“** + значка в списъка
- **Обща справка** за всички мостри (общ брой, в процес, завършени, просрочени) с печат/PDF
- Индикатор за напредък по операциите
- Автоматично запазване при всяка промяна
- Експорт / Импорт на всички данни (JSON) за резервно копие и пренос
- Печат / запис като PDF на отделна мостра

## Файлове

| Файл | Предназначение |
|------|----------------|
| `index.html` | Структура и интерфейс |
| `styles.css` | Стилове |
| `app.js` | Логика и съхранение |

## Важно за данните

Данните се пазят само в браузъра на конкретния компютър. Изтриване на
историята/кеша на браузъра може да ги изтрие. Прави редовно **Експорт** като
резервно копие.
